# Website bán laptop bằng ASP.NET

## Mô tả
Đây là đồ án xây dựng website bán laptop sử dụng ASP.NET Core MVC.  
Hệ thống hỗ trợ hiển thị danh sách sản phẩm, xem chi tiết sản phẩm, tìm kiếm và quản lý dữ liệu laptop.

## Công nghệ sử dụng
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML, CSS, Bootstrap, Script
- Visual Studio 2022

## Chức năng chính
- Hiển thị danh sách laptop
- Xem chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Quản lý sản phẩm
- Kết nối cơ sở dữ liệu SQL Server

## Cấu trúc chính
- `Controllers/`
- `Models/`
- `Views/`
- `Data/`
- `wwwroot/`

## Hướng phát triển
- Đăng nhập / đăng ký
- Giỏ hàng
- Đặt hàng
- Quản lý đơn hàng
